# Green Light Reaction

A simple browser-based reaction game where players watch a circle turn green and estimate how long it stayed green.

## Features
- Single-player mode
- Two-player mode
- Simple score feedback and winner detection

## How to Play
1. Open index.html in your browser.
2. Choose a game mode.
3. Start the round and enter your guess when the light turns off.
